# Automatic primer design
## Requirement
- [Z3](https://github.com/Z3Prover/z3)

## How to use
Install Z3.
```
git clone https://github.com/Z3Prover/z3.git
cd z3/
python scripts/mk_make.py --python
cd build; make
sudo make install
```

Install flask.
```
pip install flask
```

Start server.
```
python server.py
```
